## 2. 구현 과정
1) 레퍼런스 코드로 스켈레톤 코드 완성
2) 기능별로 나눈 클래스를 기준으로 스켈레톤 코드 리팩토링
3) 입력값에 대한 에러처리
4) https://brunch.co.kr/@javajigi/8 참고하여 리팩토링

## 3. 느낀점
- 확실히 클론코딩이 빠르게 배우기 좋은거 같음
- java collection을 잘 다룰 필요성을 느낌
